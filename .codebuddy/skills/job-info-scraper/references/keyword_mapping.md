## 岗位描述关键词匹配规则

### 规则1: 核心关键词匹配
如果岗位描述中包含任何一个核心关键词，则认为岗位匹配。

示例:
- "负责音乐活动策划和执行" → 匹配 "音乐活动策划"
- "招聘经纪人助理一名" → 匹配 "经纪人助理"

### 规则2: 组合关键词匹配
有些岗位需要组合多个关键词才能准确判断。

示例:
- "音乐" + "运营" → 音乐运营
- "演出" + "策划" → 演出策划
- "活动" + "执行" → 活动策划执行

### 规则3: 优先级判断
如果一个岗位匹配多个类型，按照以下优先级判断：

1. 如果包含"经纪人"、"经纪助理" → 艺人经纪类
2. 如果包含"演出策划"、"演出执行"、"演出统筹" → 演出策划/执行类
3. 如果包含"音乐运营"、"音乐内容营销" → 音乐运营类
4. 如果包含"自媒体"、"新媒体"、"社交媒体" → 新媒体/自媒体运营类
5. 如果包含"活动策划" → 活动策划/宣传类

## 筛选逻辑建议

### 第一轮：关键词筛选
- 必须包含至少一个核心关键词
- 排除不相关关键词（如"销售"、"财务"、"技术"等）

### 第二轮：岗位类型判断
- 根据关键词组合判断岗位类型
- 记录匹配的关键词列表

### 第三轮：经验匹配度评估
- 根据用户经验背景评估匹配度
- 高度匹配岗位优先级提升

### 第四轮：公司优先级排序
- P0公司优先
- 同一优先级内按匹配分数排序
